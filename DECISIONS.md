## ADR-1 — No static export (output: export removed)

**Status:** Accepted
**Date:** 2026-07-07

### Context

DevOS Control Center needs API routes for future live data integration.

### Decision

Use standard Next.js App Router without output: export. Vercel handles Node.js runtime deployments natively.

### Consequences

Cannot serve from CDN edge without server runtime. QA agent needs dev server start instead of static server.

## ADR-2 — Build-time data (data.ts) for MVP

**Status:** Accepted
**Date:** 2026-07-07

### Context

Live API integrations (Vercel, GitHub, Analytics) require tokens and would significantly delay MVP.

### Decision

Bake all data as TypeScript constants in data.ts. Refresh on each deploy. Replace with API calls in a future sprint.

### Consequences

Dashboard is "live as of last deploy." CEO must redeploy to get fresh data. Acceptable for current stage.

---

## ADR-3 — Dark theme throughout

**Status:** Accepted
**Date:** 2026-07-07

### Context

DevOS brand is dark (#0a0a0a, #111827 cards). Control Center should feel like the same product family.

### Decision

Dark theme as default with no light mode. Not a public product so no accessibility mandate for theme switching.

### Consequences

Simpler CSS — no theme toggle logic needed.
